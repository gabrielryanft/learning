Operadores numericos JS

operadores binarios: (Nessa ordem, de cima p/ baixo)

  1. Qualquer uma deles entre parenteses: (1 - 1).
  2. 1 ** 1
  3. 1 % 1, 1 * 1, 1 / 1
  4. 1 + 1, 1 - 1

binarios pq precisam de dois operandos(os operandos são oque os operadores usam p/ funcionar).

Se uma variavel for somada, multiplicada e etc (n = n - 3), por ela mesma, dá p/ fazer isso:

  • n **= 2
  • n %= 2
  • n *= 2
  • n /= 2
  • n += 2
  • n -= 2

e se for uma variavel mais ou menos um, dá para fazer:

  • ++n(Pré-incremento) ou n++(Pós-incremento)
  • --n(Pré-incremento) ou n--(Pós-incremento)

Tem diferença entre um e outro ("--n e ++n" e "n-- e n++" )
