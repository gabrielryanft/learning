Sans-serif

Uma fonte sem serifa. Sans-serif não é uma fonte. mas uma expecificação. diz que si quer uma fonte sem(sans)-serif(serifa).

Serif

Uma fonte serifa. serif não é uma fonte. mas uma expecificação. diz que si quer uma fonte serif(serifada).

Monospace

Uma fonte monospace. monospace não é uma fonte. mas uma expecificação. diz que si quer uma fonte monospace(todas as letras tem o mesmo tamanho).

Cursive

Uma fonte cursive. cursive não é uma fonte. mas uma expecificação. diz que si quer uma fonte cirsiva.
